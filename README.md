 📘 LearnEdge – Online Learning Platform

## 📌 What the project does

LearnEdge is a **static online learning platform** that provides users with information about courses, the platform, and ways to get in touch.
It includes separate pages for Home, About, Courses, Contact, Signup, and Login, built using **pure HTML, CSS, and JavaScript**.

---

## 💡 Why the project is useful

This project demonstrates how to:

* Build a **multi-page website** from scratch
* Use **clean and semantic HTML**
* Style pages using **modern CSS**
* Add basic interactivity with **vanilla JavaScript**
* Deploy a frontend project using **GitHub and Netlify**

It is especially useful for **beginners learning web development** and for interview or practice projects.

---

## 🚀 How users can get started with the project

### Option 1: Run locally

1. Download or clone the repository
2. Open `index.html` in any modern web browser
3. Navigate through the pages using the navbar

### Option 2: View live website

The project is deployed on Netlify and can be accessed online.
